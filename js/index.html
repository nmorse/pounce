Pounce

Pounce is a 'forth', 'joy' and 'cat' derivative programming language, inspired in part by each. Pounce returns the accessability of this kind of development language back to some modern environments/platforms:

  • the Web -- In your browser you can try it out, see examples and do quite advanced things with Pounce.
  • server-side -- Your server or any computer can run Pounce (either interpreted in Python or compiled to C# or other server-side languages).
  • microprocessors -- From Arduino to M0-4 processors You have choices of running Pounce interpretors written in 'C' or if you can run python, you can try the Pounce environment written in Python.

Pounce is a stack-based, concatenative language that uses post-fix notation. Okay this different (really different) from all the main-stream laguages that you know. Pounce is not new, its really just a remix of languages invented over the last 40 years. but it does not look or feel old. These languages have aged well. In part because of simplicity, which has been lost in most all other languages.

Simplicity was the primary motivation behind all of these languages. First of all: These languages have almost a complete lack of syntax, so that reduces what you will have to learn. Second, initially every thing will seem backwards or reversed. Post-fix notation or (or RPN) is used to write almost all code. Yeup, post-fix is strange and for the first ten minutes it will seem unneccasary, but latter you will see some very legitimate reasons for its use, we promice.

Of Course 'Backwards' is in the eye of the beholder, Looking at these two program snipits of a bowling game, which seems more 'backwards' to you?

  roleBallAt(Pins.set(TEN_PINS));
  
  TEN_PINS Pins.set roleBallAt

The first is similar to your usual fimiliar programming langage. The verbs (or actions) come first and the nouns come second. We are accustumed to this, but think about running a real bowling ally and what it takes to execute this code. It would be backwards to start at the left of the script and roll a ball at pins that had not been set up yet. The second example is in post-fix notation and it starts with the nouns and ends with verbs. It reads from left to right, with ten pins, set them up and then role a ball at them. It takes some getting used to, but the question of which one is backwards, may flip for you.
